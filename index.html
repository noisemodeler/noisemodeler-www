In-game generation

Save terrain models using a compact graph representation. Evaluation of terrain can be delayed until run-time, reducing bandwidth and storage requirements. Integrate support for procedural terrains in your game using the Noise Modeler Library.

Open source

Noise Modeler is an open-source project released under the zlib license. You are free to use the project for your closed-source game. We need your help growing and improving. Issues and suggestions are welcome in our Github repository.
